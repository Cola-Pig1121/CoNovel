# Character Design Skill - 角色设计技能

## 触发条件
Character Designer / Character Intelligence Agent 工作时加载。

## 核心三角

- **欲望(Want)** — 外在追求的目标
- **需要(Need)** — 内在缺失的东西
- **冲突(Conflict)** — 欲望和需要的矛盾

## 生存规则集

1. 定义核心性格标签
2. 基于标签设计10条行为规则
3. 在场景中测试规则一致性
4. 设计打破规则的事件 → 创造成长弧光

## 成长弧光五阶段

隐藏生存 → 试探锋芒 → 底线被触 → 主动出击 → 终极蜕变

## 反派三层金字塔

- 底层(认知型) → 早期练手
- 中层(势力型) → 中期强敌
- 顶层(终极boss) → 贯穿全书

## 细节记忆点

口头禅 + 随身物品 + 矛盾习惯 → 符合背景 → 适当频率
