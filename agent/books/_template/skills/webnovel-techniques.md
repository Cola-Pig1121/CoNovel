# Web Novel Techniques Skill - 网文写作技法库

## 触发条件
Writer / Architect Agent 工作时加载。

## 核心技法

### 爽点设计
压制(被看不起) → 积蓄(暗中准备) → 反转(实力展现) → 爽(读者满足)

### 章末钩子
章末200字内：新信息/危机/转折 → 读者必须翻页

### 黄金三章
第1章: 主角+困境+悬念
第2章: 金手指/能力展示
第3章: 第一个爽点+章末钩子

### 信息喂投法
每次只传递1条关键信息 → 用场景/对话/行动承载

### 身份反差法
普通身份 + 隐藏能力 → 第一章暗示 → 逐步揭露

### 刺激-反应法
刺激(具体事件) + 反应(身体动作 + 内心独白)

### 角色目标交织
主角目标A + 配角目标B → 行动交叉 → 剧情推进

### 双机位视角
全景航拍(环境) ↔ 第一人称肩拍(主角视角)
