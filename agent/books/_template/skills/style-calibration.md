# Style Calibration Skill - 风格校准技能

## 触发条件
Style Analyzer / Editor Agent 工作时加载。

## 风格指纹维度

- 句长分布（平均句长、短句比例、长句比例）
- 段落节奏（段落长度、对话比例、描写比例）
- 对话风格（对话长度、标签频率、动作穿插）
- 词汇偏好（词汇多样性、正式度、口语化程度）
- 情感强度（情感表达方式、感叹号密度）

## 校准流程

1. 提取当前章节风格指纹
2. 与风格锚点对比
3. 计算偏差
4. 标记需要调整的维度
5. 生成校准建议

## 风格锚点来源

- 用户在风格Tab配置的叙事视角、句式风格、词汇水平
- 参考小说分析得出的风格特征
- 历史章节的聚合风格指纹
