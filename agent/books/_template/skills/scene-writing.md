# Scene Writing Skill - 场景描写技能

## 触发条件
Writer Agent 创作具体场景时加载。

## 三维编织法

每个场景由三个维度交织：

### 发生 (Occurrence)
发生了什么（1-2句，具体细节）

### 感知 (Perception)
主角注意到了什么（视觉/听觉/触觉/嗅觉）

### 反应 (Response)
身体如何反应（动作/表情/内心）

## 段落结构

| 类型 | 字数 | 用途 |
|------|------|------|
| 厚段 | 200-500 | 多层动作、对话、心理 |
| 中段 | 90-180 | 承接场景、转换视角 |
| 薄段 | 30-90 | 短促判断、转折钉 |
| 一句成段 | 8-30 | 冷判断或收束 |

连续性禁令：禁连续3段<90字 / 禁连续2段一句成段 / 禁连续3段厚段

## 句式节奏

- 长句(30-80字)为主，短句(<15字)仅在关键判断处
- 一段内短句≤2个
- 全文至少3处<15字短句作为顿挫

## 对话规则

- 动作穿插对话（action beats）
- 减少"他说""她问"
- 对话潜台词：说的不是想的
