# Outline Builder Skill - 大纲构建技能

## 触发条件
Architect Agent 构建大纲时加载。

## 双套大纲法

### 宏观大纲
用固定句式概括：他是…；他遇到… / 他原本…；因为…导致… / 他现在…；他希望…

### 微观大纲
1. 列元素（冲突、背景、伏笔、动作等）
2. 按重要性排序
3. 先写最重要的两个
4. 卡住时换下一个

## 点线面体扩展

点(创意) → 线(故事线) → 面(完整情节) → 体(细节充实)

## 章节蓝图模板

每章包含：
- 章节号、标题、摘要
- 核心事件
- 目标情感
- 钩子列表
- 参与角色
- 地点
- 目标字数
- 支线进度

## 节奏控制

- 高潮章节后安排缓冲
- 连续高潮不超过3章
- 每章至少1个章末钩子
- 每3章至少1个悬念推进
