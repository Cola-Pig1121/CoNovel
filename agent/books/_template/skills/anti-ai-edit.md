# Anti-AI Edit Skill - 去AI味编辑技能

## 触发条件
De-AI Editor Agent 工作时加载。

## 核心原则

修补，不重写。每个命中项做一次最小修补。
作者优先。"更干净" vs "更像作者" → 以后者为准。
回滚闸门。修补后段落变薄/语调变平 → 回滚。

### 语姿类(4)
