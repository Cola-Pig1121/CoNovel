# Scene Writing Skill - 场景写作技能

## 触发条件
当Agent需要创作具体场景、描写动作、构建氛围时加载此技能。

## 技能内容

### 三维编织法 (Three-Dimension Weaving)

每个场景由三个维度交织而成：

#### 1. 发生 (Occurrence)
发生了什么（1-2句，具体细节）：
- 核心动作/事件
- 具体的时间、地点
- 参与的角色

#### 2. 感知 (Perception)
主角注意到了什么（感官细节）：
- 视觉：看到的景象
- 听觉：听到的声音
- 触觉：感受到的触感
- 嗅觉：闻到的气味
- 味觉：尝到的味道（如有）

#### 3. 反应 (Response)
身体如何反应（具体动作）：
- 身体动作
- 面部表情
- 内心活动
- 情感变化

### 三维编织示例

**原始版本（扁平）：**
> 林天看到了敌人，感到很愤怒，冲了上去。

**三维编织版本（立体）：**
> 黑影从屋檐掠过，带起一阵腥风。（发生）
>
> 林天瞳孔骤缩，那熟悉的气息——是血煞宗的人。（感知）
>
> 他握紧手中长剑，指节因用力而发白，牙关紧咬。（反应）

### 场景类型模板

#### 动作场景
```
[环境铺垫] - 1-2句，营造紧张氛围
[动作发生] - 核心动作，具体有力
[感官细节] - 视觉、听觉、触觉
[身体反应] - 主角的本能反应
[节奏变化] - 快慢交替
```

#### 对话场景
```
[场景设定] - 交代对话环境
[对话内容] - 符合角色性格
[动作穿插] - 动作打断对话
[心理活动] - 角色内心想法
[情感变化] - 对话带来的情感波动
```

#### 环境描写
```
[整体印象] - 第一眼看到的
[细节观察] - 主角注意到的细节
[感官体验] - 身体感受到的
[情感投射] - 环境唤起的情感
```

### 节奏控制

#### 快节奏
- 短句为主
- 动作密集
- 感官聚焦
- 段落短小

#### 慢节奏
- 长句为主
- 细节丰富
- 内心描写
- 段落较长

#### 节奏交替
- 高潮前铺垫
- 高潮时加速
- 高潮后缓冲

### 对话写作

#### 对话原则
- 符合角色身份和性格
- 有潜台词（说的不是想的）
- 有动作穿插（不要纯对话）
- 有情感变化

#### 对话标签
- 少用"他说"、"她问"
- 用动作代替标签
- 标签要简洁

#### 示例
**不好：**
> "你好。"林天说。
> "你好。"小红说。
> "今天天气不错。"林天说。

**好：**
> 林天推开门，小红正坐在窗边。
>
> "来了？"她头也没抬，手指轻叩桌面。
>
> "嗯。"林天在她对面坐下，目光扫过桌上的茶杯，"等很久了？"
>
> "刚到。"小红终于抬起头，嘴角微扬，"还是说，你希望我等很久？"

### 注意事项
1. 具体胜过抽象 - "他握紧了剑"比"他很愤怒"好
2. 感官要丰富 - 不要只写视觉
3. 节奏要变化 - 不要全程一个速度
4. 细节要服务情节 - 不要为描写而描写
