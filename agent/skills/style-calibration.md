# Style Calibration Skill - 风格校准技能

## 触发条件
当Agent需要校准写作风格、分析风格指纹、确保风格一致性时加载此技能。

## 技能内容

### 风格指纹提取

#### 1. 词汇特征
- **词汇多样性** - 词汇重复率
- **词汇偏好** - 常用词汇类型
- **成语使用** - 成语频率和类型
- **口语化程度** - 书面语/口语比例

#### 2. 句式特征
- **平均句长** - 句子平均字数
- **句长分布** - 长短句比例
- **句式类型** - 陈述句/疑问句/感叹句比例
- **复合句比例** - 简单句/复合句比例

#### 3. 段落特征
- **平均段落长度** - 段落平均字数
- **段落结构** - 单一主题/多主题段落
- **段落过渡** - 过渡方式（自然/突兀）

#### 4. 对话特征
- **对话比例** - 对话占全文比例
- **对话长度** - 平均对话长度
- **对话标签** - 标签使用频率
- **动作穿插** - 动作打断对话的频率

#### 5. 情感特征
- **情感强度** - 情感描写的强烈程度
- **情感类型** - 喜/怒/哀/乐的比例
- **情感节奏** - 情感变化的频率

### 风格一致性检查

#### 检查维度
1. **词汇一致性** - 词汇使用是否与风格锚点一致
2. **句式一致性** - 句式模式是否与风格锚点一致
3. **段落一致性** - 段落结构是否与风格锚点一致
4. **对话一致性** - 对话风格是否与风格锚点一致
5. **情感一致性** - 情感表达是否与风格锚点一致

#### 偏差处理
- **轻微偏差** - 记录，不干预
- **中度偏差** - 警告，建议调整
- **严重偏差** - 干预，强制校准

### 风格进化追踪

#### 进化指标
- **稳定性** - 风格波动程度
- **成长性** - 风格改进程度
- **多样性** - 风格变化丰富度

#### 进化策略
- **保持优势** - 保持已经很好的风格特征
- **改进弱点** - 改进得分较低的维度
- **尝试创新** - 在保持稳定的基础上尝试新风格

### 风格锚点管理

#### 锚点来源
- **参考文本** - 从参考文本提取风格
- **历史章节** - 从已完成章节提取风格
- **用户指定** - 用户指定的目标风格

#### 锚点更新
- **定期校准** - 每10章校准一次
- **重大变化后** - 风格发生重大变化后重新校准
- **用户干预后** - 用户手动调整后更新

### 注意事项
1. 风格校准要尊重作者的个人风格
2. 不要把所有作品都校准成同一个风格
3. 风格进化要渐进，不能突变
4. 风格一致性不是风格单一化
