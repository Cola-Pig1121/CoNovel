# Character Design Skill - 角色设计技能

## 触发条件
当用户请求设计角色、创建人物、构建关系图谱时加载此技能。

## 技能内容

### 角色设计方法论

#### 1. 核心三角
每个主要角色需要三个核心要素：
- **欲望（Want）** - 角色外在追求的目标
- **需要（Need）** - 角色内在缺失的东西
- **冲突（Conflict）** - 欲望和需要之间的矛盾

#### 2. 性格层次
- **表层性格** - 角色展示给外界的样子
- **深层性格** - 角色真实的内在
- **性格弧线** - 角色性格如何随故事发展

#### 3. 背景要素
- **出身** - 角色的出身背景
- **经历** - 影响角色的关键经历
- **创伤** - 角色的心理创伤（如有）
- **执念** - 角色的执念或信念

### 关系图谱设计

#### 关系类型
- **血缘** - 家族、师徒
- **情感** - 爱情、友情、仇恨
- **利益** - 合作、竞争、背叛
- **理念** - 同道、对立

#### 关系强度
- **核心关系** - 影响主线剧情
- **重要关系** - 影响角色决策
- **次要关系** - 丰富角色世界

#### 关系发展
- 关系要自然演进，不能突变
- 重大关系变化需要铺垫
- 关系变化要推动剧情

### 角色一致性检查

#### 检查维度
1. **性格一致性** - 角色行为是否符合性格设定
2. **动机一致性** - 角色行为是否符合其目标
3. **能力一致性** - 角色表现是否符合其能力设定
4. **成长一致性** - 角色成长是否符合弧线轨迹

#### 常见问题
- 角色突然变聪明/变傻
- 角色能力忽高忽低
- 角色性格前后矛盾
- 角色关系突变无铺垫

### 注意事项
1. 主要角色要有深度，次要角色要有特色
2. 角色要有人味，不要完美或脸谱化
3. 角色关系要复杂但不混乱
4. 角色成长要有代价，不能白嫖
