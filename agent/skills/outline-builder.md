# Outline Builder Skill - 大纲构建技能

## 触发条件
当用户请求创建大纲、规划章节、设计情节时加载此技能。

## 技能内容

### 大纲构建方法论

#### 1. 自顶向下法
从整体到局部，逐步细化：
- 确定小说主题和核心冲突
- 规划卷/章结构
- 为每卷设计主线进展
- 为每章创建详细蓝图

#### 2. 冲突驱动法
以冲突为核心构建情节：
- 核心冲突（贯穿全书）
- 卷冲突（每卷解决一个阶段）
- 章冲突（每章的小冲突）
- 场景冲突（场景内的微冲突）

#### 3. 角色驱动法
以角色成长为核心构建情节：
- 角色起点状态
- 角色成长目标
- 阻碍角色成长的障碍
- 角色克服障碍的关键时刻

### 节奏控制原则

#### 三幕结构
- **第一幕（25%）** - 建立常态，引入冲突
- **第二幕（50%）** - 冲突升级，主角挣扎
- **第三幕（25%）** - 高潮决战，解决冲突

#### 张弛交替
- 高潮章节后安排缓冲章节
- 连续高潮不超过3章
- 缓冲章节也要有小钩子

#### 钩子密度
- 每章至少1个章末钩子
- 每3章至少1个悬念推进
- 每卷至少1个重大反转

### 注意事项
1. 大纲是指导，不是枷锁 - 允许执行中微调
2. 保持整体视角 - 不要为单章精彩破坏整体节奏
3. 伏笔要有始有终 - 不要挖坑不填
4. 角色要有人味 - 不要为剧情让角色做不合理的事
